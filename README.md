# RAG PDF Chatbot (OpenAI + LangChain + ChromaDB)

This project is a Retrieval-Augmented Generation (RAG) chatbot that lets you upload a PDF and ask natural-language questions about its content. The system retrieves the most relevant text chunks using embeddings and answers using an LLM.

## Features
- Upload ANY PDF
- Automatic text loading + splitting
- ChromaDB vector store
- RAG retrieval pipeline
- Streamlit UI for chatting with documents

## Technologies Used
- Streamlit
- LangChain
- ChromaDB
- OpenAI GPT-4o-mini (API)
- Python 3.10
